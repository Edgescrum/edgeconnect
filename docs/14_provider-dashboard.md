# 14. 事業主向けダッシュボード

## 概要
事業主がログイン後に最初に表示されるホーム画面。予約状況の把握と各管理画面への導線を提供する。

## 現状
- トップページ（`/`）でロールに応じた出し分けを実装済み
- 予約統計は仮値（0）を表示中
- 管理メニューへのリンク（予約ページ・QRコード・プロフィール編集）を表示
