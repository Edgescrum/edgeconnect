# 19. SNSシェアテンプレート

## 概要
事業主が自分の予約ページをSNS/LINEで共有しやすくするテンプレート文面とシェアボタン。

## 実装箇所
`/provider/qrcode` ページ（既存のURL/QRコード画面に追加セクション）

## シェア方法

| 方法 | 実装 |
|------|------|
| URLコピー | テンプレ文+URLをクリップボードにコピー |
| LINEシェア | `https://line.me/R/share?text={エンコード済みテキスト}` |
| Web Share API | 対応ブラウザで `navigator.share()` を使用。非対応時はURLコピーにフォールバック |

## テンプレート文面

```
{屋号}のご予約はこちらから！
LINEで簡単に予約できます。
{予約ページURL}
```

- 事業主が文面を編集可能にする（テキストエリアにデフォルト文面をセット）
- URLは自動挿入（編集不可）

## UIイメージ

```
┌──────────────────────────────┐
│ SNSでシェア                    │
│                                │
│ ┌─────────────────────────┐   │
│ │ {屋号}のご予約はこちらから！│   │
│ │ LINEで簡単に予約できます。 │   │
│ │ https://...              │   │
│ └─────────────────────────┘   │
│                                │
│ [LINEでシェア] [コピー] [シェア]│
└──────────────────────────────┘
```

## 注意事項
- LINE内ブラウザでは `navigator.share` 非対応 → フォールバック必須
- `<a download>` もLINE内ブラウザ非対応のため、シェアはURL scheme ベースで実装
