# 法的文書の整備

## 概要
Stripe導入前（V2リリース前）に必要な法的文書を整備する。

## 必要文書

### 1. プライバシーポリシー
個人情報保護法に基づき、収集する個人情報と利用目的を明示。

#### 記載すべき個人情報
| 情報 | 取得元 | 用途 |
|------|--------|------|
| LINE ID | LINEログイン | 認証 |
| 表示名 | LINEプロフィール | 表示 |
| 氏名 | 予約時入力/設定 | 予約管理 |
| 電話番号 | 予約時入力/設定 | 事業主連絡用 |
| 性別 | 設定（任意） | 分析 |
| 生年月日 | 設定（任意） | 分析 |
| 予約履歴 | システム | サービス提供 |
| アンケート回答 | 回答 | 改善・口コミ |
| 決済情報 | Stripe | 課金（PeCo側では保持しない） |

#### 記載項目
- 収集する個人情報の種類
- 利用目的
- 第三者提供（LINE/Stripe/Supabaseへの情報連携）
- 個人情報の管理・保護方法
- 開示・訂正・削除の請求方法
- Cookie・アクセスログの取り扱い
- 問い合わせ窓口

### 2. 利用規約
サービス利用条件の明示。

#### 記載項目
- サービスの定義と範囲
- アカウント登録・管理
- 禁止事項
- 口コミ公開のルール（公開許可・NGワード・非表示基準）
- サービス料金とプラン（ベーシック/スタンダード/プロ）
- トライアルと自動課金
- 解約・退会
- 免責事項
- 規約変更の通知方法
- 準拠法・管轄裁判所

### 3. 特定商取引法に基づく表記
月額課金サービスのため必須。

#### 記載項目
- 販売業者名（会社名/屋号）
- 代表者名
- 所在地
- 連絡先（メール/電話）
- 販売価格（ベーシック¥500/月、スタンダード¥980/月、プロ¥1,980/月）
- 支払方法（クレジットカード/Stripe）
- 支払時期
- サービス提供時期
- 返品・キャンセルポリシー
- 解約方法

## 表示場所
- フッターにリンク追加（プライバシーポリシー / 利用規約 / 特商法表記）
- 事業主登録フロー内で利用規約への同意チェック
- Stripe決済前に特商法表記へのリンク

## 作成方針
- テンプレートベースで自作
- 運営会社情報は定款等を元に記入
- Stripe導入前に完成させる
