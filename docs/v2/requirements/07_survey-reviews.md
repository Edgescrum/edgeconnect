# アンケート・ユーザー評価

## 概要
予約完了後にアンケートを自動送信し、事業主の改善活用と口コミ公開を実現する機能。

### コンセプト
- アンケートは事業主の改善ツール（非公開）
- 口コミはユーザーの判断材料（公開・許可制）
- カテゴリ別ベンチマークで業界比較可能

---

## 基本仕様

| 項目 | 内容 |
|------|------|
| 送信タイミング | 予約終了時刻の1時間後にLINE通知で自動送信 |
| 回答期限 | 7日間 |
| 回答ページ | `/survey/[bookingId]` |
| リマインド | なし |

### 追加テンプレート設問（スタンダード・プロのみ）

- 業界別テンプレート設問集から最大3つ選択
- 文言は事業主が微修正可能
- 回答形式は5段階固定（分析統一のため）
- ベンチマーク比較可能（同じテンプレートを使う事業主同士）

## 口コミ

### 公開フロー
1. お客さまが設問6に回答
2. 「このコメントを公開してもよいですか？」チェックボックスON
3. NGワードフィルタ通過 → 公開
4. NGワード検出 → 非公開（お客さまに通知なし）

### NGワードフィルタ
- 正規表現ベース（AIなし）
- 暴言・差別用語等の明らかなNGワードをブロック
- すり抜けた場合は事業主が非表示で対応

### 管理
- 事業主: 管理画面で非表示にできる（理由不要）
- お客さま: 自分の口コミを削除可能（編集不可）
- 非表示にしても分析データには残る（スコア操作不可）

### 表示場所
- 事業主プロフィール（`/p/[slug]`）にレビューセクション
- `/p/[slug]/reviews` 専用ページ（全件表示）
- `/explore` 一覧カードにCSAT平均★+件数を表示

## 分析ダッシュボード（`/provider/analytics`内）

### ベーシック
- CSAT平均スコア（数値のみ）
- 直近の回答一覧

### スタンダード・プロ

#### サマリー
- CSAT平均スコア（前月比）
- 各ドライバー平均スコア（接客・品質・価格）
- 回答率

#### トレンド
- CSAT月次推移グラフ（過去6ヶ月）
- ドライバー別月次推移

#### 属性別マトリックス

| 軸 | クロス集計 |
|---|-----------|
| 性別 × CSAT | 男性★4.5 / 女性★4.1 / その他★4.3 |
| 年代 × CSAT | 20代★4.6 / 30代★4.2 / 40代★4.0 |
| 新規/リピート × CSAT | 新規★4.5 / リピート★3.8 |
| メニュー × CSAT | カット★4.3 / カラー★4.1 / パーマ★3.9 |
| 性別 × ドライバー | 男性: 接客4.5/品質4.3 / 女性: 接客4.0/品質4.2 |
| 年代 × ドライバー | 同上 |
| 新規/リピート × ドライバー | リピーターの価格納得感が低い等 |
| LTVセグメント × CSAT | 優良顧客★4.5 / 休眠顧客★3.8 |
| LTVセグメント × ドライバー | 高LTV顧客は品質重視、低LTVは価格重視 等 |

#### 業界ベンチマーク比較
- 同カテゴリ事業主との匿名比較
- 最低5事業主のデータがある場合に表示
- 比較項目: CSAT・各ドライバー・リピート率・平均LTV
- 表示形式: 自分 vs 業界平均（差分表示、警告マーク付き）

#### ワードクラウド
- 自由記述の頻出キーワード（kuromoji.js）

#### AI改善提案（プロのみ）
- 月次バッチで全回答をAI分析
- 「リピーターの満足度が下がっています。価格の見直しを検討してください」等

## 属性情報（V2で`/settings`に追加）

| 項目 | 種類 | 必須 |
|------|------|:---:|
| 性別 | 選択（男性/女性/その他/回答しない） | 任意 |
| 生年月日 | 日付 | 任意 |

- 初回ログイン時に設定ページへ誘導
- 生年月日から利用日時点の年齢を自動算出
- 分析時にDB情報（新規/リピート、メニュー）と合わせて属性別マトリックスに活用
