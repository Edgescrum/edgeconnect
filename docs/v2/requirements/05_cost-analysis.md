## 事業主数別アクションリスト

### 🟢 0〜30事業主
- [ ] LINEコミュニケーションプラン（無料200通）で運用
- [ ] Vercel Hobby / Supabase Free
- [ ] **月間通知数を監視**（200通に近づいたらアラート）

### 🟡 30事業主到達時（通知200通超過見込み）
- [ ] **LINEライトプラン（¥5,000/月）に切り替え**
- [ ] 通知送信数のダッシュボード監視を開始

### 🟡 100事業主到達時

- [ ] DB容量・クエリパフォーマンスの監視強化
- [ ] CDN・画像最適化の検討

## 通知コスト最適化のヒント

- 1予約あたり3通（確定通知×2 + リマインダー）が基本
- リマインダーをスタンダード以上限定にすれば通知数を1/3削減可能 → **ただしV2ではベーシックにもLINE通知を含める方針のため不可**
- 将来的にプッシュ通知（PWA）を併用すればLINE通知数を削減可能
