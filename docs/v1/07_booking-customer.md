# 07. 予約受付 — お客さん側（優先度1）

## 概要
お客さんがLIFF上でサービスを選択し、空き日時から予約を確定する。

## 仕様
| 機能 | 詳細 |
|------|------|
| 空き日時の閲覧 | 事業主の営業時間・既存予約・インターバルを考慮した空き枠を表示 |
| 予約作成 | サービス選択 → 日時選択 → 即時確定 |
| 予約確認通知 | 確定時にLINE Flex Messageで自動送信 |
| リマインダー通知 | 予約前日にLINE Flex Messageで自動送信 |
| 予約キャンセル | キャンセル期限内のみLIFFページからキャンセル可能 |
| 予約履歴の確認 | 自分の予約一覧を閲覧 |

## 予約フロー
```
① 事業主プロフィールページにアクセス
② サービスメニューを選択
③ 空き日時カレンダーから日時を選択
④ 予約内容確認画面で確定
⑤ bookingsテーブルにINSERT（排他制御）
⑥ 予約確認通知をLINE Flex Messageで送信（お客さん＋事業主）
```

## キャンセルポリシー
- キャンセル期限は `services.cancel_deadline_hours` で判定
- 期限を過ぎた場合はキャンセルボタンを非表示・無効化
- 期限後のキャンセル希望は事業主とのLINE直接やりとりに委ねる
